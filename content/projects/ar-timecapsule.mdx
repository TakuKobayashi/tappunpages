## Overview

現実世界の特定座標に AR メッセージを「埋める」プラットフォーム。
ユーザーはその場所を訪れることで初めてメッセージを見ることができる。

## Key Features

- **ARCore / ARKit 対応** — Unity AR Foundation でクロスプラットフォーム
- **ジオハッシュ近傍探索** — 効率的な位置ベース検索
- **カーソルベースページネーション** — 大量データの無限スクロール
- **R2 メディアストレージ** — 画像・動画のタイムカプセル

## Stack

- **AR Client**: Unity 2022, AR Foundation, ARCore, ARKit
- **Backend**: Cloudflare Workers, Hono, Drizzle ORM
- **Storage**: Cloudflare D1, R2
- **Location**: Geohash algorithm
